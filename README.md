## Setup
### Ubuntu 20.04
Install Ubuntu 20.04 on Raspberry Pi

#### Build

```
cd ~/
git clone https://github.com/ku-gcl/gcl-pendulum.git
cd ~/gcl-pendulum
source Tools/setup.sh
```
